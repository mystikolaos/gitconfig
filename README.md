# Git config

A set of Bash scripts for setting up my Git configuration. The scripts are tailored to my personal preferences, but they are simple enough to be easily adapted to suit your own needs, so help yourself.

## Usage

Just run the scripts in any console supporting Bash commands. On Linux any terminal will work, on Windows use Git Bash. To create global Git configuration file, run:

    ./bin/make_gitconfig

And to create global `.gitignore` file, run:

    ./bin/make_gitignore

That's it, done!
